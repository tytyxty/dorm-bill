# 宿舍水电费分摊计算器

## 项目简介

这是一个用于解决宿舍水电费分摊问题的 Python 程序。支持宿舍成员信息的添加、修改、删除，录入每月水费和电费金额，自动计算每人应分摊的金额，同时支持查看历史记录和记录详情。

## 功能列表

### 成员管理
- 查看所有成员
- 添加成员
- 修改成员姓名
- 删除成员

### 费用录入
- 录入每月水费金额
- 录入每月电费金额
- 自动计算当月总费用
- 防止重复录入同月份记录

### 分摊计算
- **平均分摊**：按人数平均分配
- **按人数比例分摊**：支持自定义权重（如多人同住一间房权重为2）

### 历史查询
- 查看所有历史费用记录
- 查看单条记录详细内容
- 每条记录保存当时的成员快照

## 数据存储

程序使用 JSON 文件存储数据：
- `members.json`：存储宿舍成员名单
- `records.json`：存储历史费用记录

### 记录格式示例
```json
{
  "month": "2026年07月",
  "water": 100,
  "electricity": 80,
  "members": ["张三", "李四", "王五"]
}
```
