# 宿舍水电费分摊计算器
## 项目简介
这是一个用于解决宿舍水电费分摊问题的 Python 程序。可以添加/删除成员，录入每月水电费，并自动计算每人应分摊的金额，同时支持查看历史记录。
## 功能列表
- 成员管理：添加、删除、查看宿舍成员
- 费用录入：录入每月水费和电费金额
- 自动计算：根据成员人数自动计算每人分摊金额
- 历史查询：查看所有历史费用记录
## 开发环境
- Python 3
- Jupyter Notebook
## 运行方式
1. 安装 Python 3 和 Jupyter Notebook
2. 打开 `宿舍水电费分摊计算器.ipynb` 文件
3. 按顺序运行所有代码单元格
